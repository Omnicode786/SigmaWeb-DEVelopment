## 🚨 TROUBLESHOOTING

**Problem: "Cannot find module"**
Solution: Delete node_modules folder, run `npm install` again

**Problem: "Port 5173 already in use"**
Solution: Change port in vite.config.js or kill the process using port 5173

**Problem: "Canvas not rendering"**
Solution: Check WebGL support in your browser. Try different browser.

**Problem: "Styles not applying"**
Solution: Clear browser cache, rebuild Tailwind CSS

More detailed troubleshooting in SETUP_GUIDE.md
